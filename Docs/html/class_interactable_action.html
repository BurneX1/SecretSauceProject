Detailed Description

Script encargado de asignar valores a un nuevo script, para que este sea interactuable.

Para crear un objeto que realice alguna acción como respuesta a alguna interacción utilizando la clase “InteractableAction”, puede crear un nuevo script que herede las funciones de esta clase y sobrescribir el método de la interacción a como considere conveniente.
